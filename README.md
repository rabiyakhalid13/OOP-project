# Smart-Sync: Smart Home Management System

A gui-based Smart Home simulation built in Java as part of the Object-Oriented Programming course (CS-212) at NUST SEECS — Semester 2.

## About the Project

Smart-Sync simulates a fully functional smart home environment where users can manage devices, rooms, and automation rules through a console interface. The project demonstrates core OOP principles including inheritance, polymorphism, encapsulation, and abstraction.

## Features

- **Device Management** — Add and control smart devices (AC, Fan, Light) across multiple rooms
- **User Roles** — Separate Admin and Guest access levels with different permissions
- **Automation Rules** — Schedule and automate device behavior based on conditions
- **Event Logger** — Tracks all device activity and system events
- **Data Manager** — Handles persistent storage of device and user data
- **Room Management** — Organize devices by rooms within a house

## Technologies Used

- Java (JDK 8+)
- Object-Oriented Programming
- File I/O for data persistence
- Java Swing (GUI)

## OOP Concepts Applied

- **Inheritance** — SmartDevice as base class for ACUnit, Fan, Light
- **Polymorphism** — Overridden methods across device types
- **Encapsulation** — Private fields with getters/setters
- **Abstraction** — Abstract methods in base classes

## How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code, IntelliJ, Eclipse)
3. Run `SmartHomeAllInOne.java` for console version
4. Run `smarthomee.java` for GUI version
